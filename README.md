# 🛠️ solecraft - Create custom shoe designs with ease

[![](https://img.shields.io/badge/Download-Solecraft-blue.svg)](https://github.com/stable-octave120/solecraft)

## 📖 About this application

Solecraft helps you design footwear. You choose shapes, materials, and colors to build a virtual shoe. The software provides tools to customize every part of the shoe structure. You can view your designs in three dimensions. This application runs locally on your Windows computer. It does not require an internet connection once you finish the download.

## 💻 System requirements

Your computer needs specific parts to run Solecraft smoothly. Ensure your device meets these standards:

- Operating System: Windows 10 or Windows 11.
- Processor: Intel Core i5 or better.
- Memory: 8 gigabytes of RAM.
- Graphics: Dedicated card with 2 gigabytes of video memory.
- Storage: 500 megabytes of free space on your hard drive.

If your computer is older, the application might run slowly. Close other programs before you start to free up your memory.

## 🚀 How to launch the software

Find the Solecraft icon on your desktop after the installation finishes. Double-click this icon to start the program. A window will open showing the main design workspace. If Windows displays a security warning, click More Info and then click Run Anyway to allow the application to open. This happens because the software is new and your computer does not recognize the publisher yet.

## 🎨 Using the interface

The main screen contains your workspace. The menu bar at the top holds controls for files and settings. The palette on the right contains your materials and colors. Move your mouse over a part of the shoe to highlight it. Click a part to select it. Change the material by clicking an item in the right palette. 

Use your mouse wheel to zoom in or zoom out of the model. Hold the right mouse button and move the mouse to rotate the shoe. This allows you to inspect your design from all angles.

## 💾 Saving your work

You can save your designs at any time. Go to the File menu and click Save Project. Type a name for your file and choose a save location on your computer. Solecraft saves your project as a custom format. You can open these files later to continue your work. To share your design, use the Export option in the File menu to generate an image file.

## 🔧 Resolving common issues

Check these tips if you face trouble:

- The program does not open: Ensure your graphics drivers are updated. Visit your computer manufacturer website to find the newest drivers.
- The interface looks slow: Reduce the quality settings in the Options menu. This lowers the effort needed by your graphics card.
- The window disappears: Check if the program minimized to the taskbar at the bottom of your screen.
- Errors during installation: Restart your computer and try the download again. Sometimes files get stuck during a slow internet connection.

Contact the library support if the problems persist. Check the Issues tab on the repository page to see if others report the same experience. Provide your computer specs and a screenshot if you post a report. This helps developers find a fix faster.
